POISKTOVAROV - это мобильное приложение для удобного и быстрого заказа товаров с доставкой на дом. Пользователь может легко найти нужный товар, ознакомиться с его описанием и оформить заказ в несколько шагов. Приложение создано для упрощения процесса покупок и экономии времени. Проект разработан студентами группы 11 ВТИ-23 - Ким Д. и Нурбек А. и ориентирован на современного пользователя, ценящего комфорт и простоту.
